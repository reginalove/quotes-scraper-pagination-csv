# Quotes Scraper with Pagination (CSV)

This Python project scrapes quotes from quotes.toscrape.com across multiple pages.

## Features

- Scrapes quotes text
- Extracts author names
- Extracts tags
- Handles pagination automatically
- Saves data into CSV file

## Technologies Used

- Python
- requests
- BeautifulSoup
- CSV

## Purpose

Built to practice web scraping, pagination handling, and structured data export.
